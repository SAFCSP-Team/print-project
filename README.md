# Print Project

### Objective

In this project, our main goal is to understand how to use print statement in C++.

### Problem

Develop a program that prints the following. 

```
Hello User,
Nice to meet you!
```

### Implementation
- Write your code in the main function.
  
```cpp
#include <iostream>

int main() {
    /*Your code here*/
    return 0;
}

```
